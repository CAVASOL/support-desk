# Support Desk

React app as Support desk and Ticketing system with MERN Stack

<img align="center" src="./supportdesk.png" width="800px" />

## Usage

Add your MONGO_URI to .env

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```
